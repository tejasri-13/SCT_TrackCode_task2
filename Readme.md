# SCT_trackcode_task2 - Stopwatch App

A modern, responsive stopwatch application built with HTML, CSS, and JavaScript.

## Features

- **Start/Pause/Reset Functionality**: Full control over the stopwatch timer
- **Lap Recording**: Record lap times while the stopwatch is running
- **Real-time Display**: Live updating time display in HH:MM:SS format
- **Responsive Design**: Works on desktop and mobile devices
- **Clean UI**: Modern gradient background with clear button styling
